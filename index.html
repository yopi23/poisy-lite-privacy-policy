👋 1. Pendahuluan

Kebijakan Privasi ini menjelaskan bagaimana Poisy Kasir ("kami", "aplikasi", atau "layanan kami") mengumpulkan, menggunakan, menyimpan, dan melindungi informasi pribadi serta data operasional usaha Anda saat menggunakan aplikasi mobile Android Poisy Kasir dan platform Web kami.

✅ Komitmen Privasi: Kami menghormati kerahasiaan data bisnis Anda. Kami TIDAK PERNAH menjual atau menyewakan data pribadi atau data transaksi toko Anda kepada pihak ketiga.

📊 2. Informasi yang Kami Kumpulkan

Untuk menyediakan layanan sistem Point of Sale (POS) yang optimal, kami mengumpulkan jenis data berikut:

a. Data Akun & Pengguna

  • Informasi Pendaftaran: Nama pengguna, alamat email, dan kata sandi yang dienkripsi.
  • Profil Pengguna: Foto profil (opsional), peran dalam toko (Admin Pemilik Toko atau Kasir/Staff).

b. Data Usaha & Toko (Tenant)

  • Informasi Toko: Nama usaha, alamat toko, nomor telepon, logo toko, dan rincian kontak bisnis.
  • Data Operasional POS: Katalog produk, harga, kategori, stok barang, data pemasok (supplier), serta data pelanggan toko.
  • Data Transaksi: Riwayat penjualan, tanggal transaksi, metode pembayaran, rincian item belanja, dan laporan keuangan toko.

c. Data Teknis & Perangkat

  • Informasi Perangkat: Model perangkat, versi sistem operasi (Android/Web), IP address, log kegagalan (crash logs), dan data performa aplikasi untuk diagnostik keamanan dan peningkatan mutu aplikasi.

⚙️ 4. Penggunaan Data

Kami menggunakan data yang dikumpulkan untuk tujuan berikut:

  • Layanan Utama: Menyediakan sistem pencatatan transaksi kasir, manajemen stok, dan laporan analisis keuangan toko Anda.
  • Sinkronisasi Multi-Perangkat: Memastikan data toko Anda tersinkronkan dengan aman antara aplikasi Android dan dasbor Web.
  • Dukungan Pelanggan: Membantu menyelesaikan kendala teknis atau pertanyaan operasional yang Anda ajukan.
  • Keamanan & Pencegahan Fraud: Memverifikasi identitas pengguna, melindungi akun dari akses tak sah, serta menjaga integritas database.

🌐 5. Layanan Pihak Ketiga & Infrastruktur Cloud

Kami dapat menggunakan penyedia layanan pihak ketiga yang terpercaya untuk mendukung operasional infrastruktur aplikasi:

  • Layanan Hosting & Cloud Database: Digunakan untuk meng-host database terenkripsi dan API server backend aplikasi. Penyedia cloud diikat oleh perjanjian kerahasiaan data yang ketat.

🚫 Kami TIDAK MENJUAL, menyewakan, membagikan, atau memanfaatkan data transaksi bisnis Anda untuk keperluan iklan pihak ketiga atau pemasaran tanpa izin eksplisit dari Anda.

🛡️ 6. Keamanan Data & Enkripsi

Kami menerapkan standar keamanan industri untuk melindungi informasi pribadi dan data transaksi Anda:

  • Enkripsi saat Transit (In-Transit): Seluruh lalu lintas data antara aplikasi Android/Web dan server kami dilindungi menggunakan protokol enkripsi aman HTTPS / TLS 1.2+.
  • Enkripsi saat Penyimpanan (At-Rest): Kata sandi pengguna diacak menggunakan algoritma hashing satu arah yang kuat (BCrypt/Argon2). Database kami dilindungi dengan proteksi akses berlapis.
  • Kontrol Akses Terbatas: Hanya pengguna terotorisasi dengan peran yang sesuai (Admin vs Staff) yang memiliki akses ke data operasional toko masing-masing.

🗑️ 7. Retensi Data & Hak Penghapusan Akun (Account Deletion)

Sesuai dengan kebijakan pengguna Google Play Store, kami menyediakan mekanisme yang jelas bagi pengguna untuk meminta penghapusan akun dan data pribadi mereka.

a. Prosedur Penghapusan Akun Pemilik Toko (Admin)

Pemilik Toko (Admin) dapat mengajukan penghapusan permanen akun beserta seluruh data tokonya dengan dua cara:

  1. Melalui Aplikasi Mobile (In-App): Buka menu Pengaturan / Profil → pilih Hapus Akun & Data Toko → Konfirmasi permintaan Anda.
  2. Melalui Permintaan Email / Layanan Pelanggan: Kirimkan permintaan resmi melalui email ke support@poisy.com dengan subjek Permintaan Penghapusan Akun - [Nama Toko/Email Admin].

b. Akun Kasir / Staff

Akun Kasir (Staff) dibuat dan dikelola di bawah akun Admin Toko. Penghapusan atau penonaktifan akun Kasir dilakukan secara langsung oleh Admin Toko melalui menu Manajemen Kasir di dalam aplikasi.

c. Kebijakan Retensi & Penghapusan Permanen Data

  • Setelah permintaan penghapusan akun diverifikasi, seluruh data pribadi, kredensial login, katalog produk, dan riwayat transaksi toko akan dihapus secara permanen dari server aktif kami dalam waktu 30 hari kalender.
  • Data cadangan (backup) yang terenkripsi akan dibersihkan secara otomatis sesuai dengan siklus pembersihan berkala server (maksimal 60 hari).

👶 8. Privasi Anak-Anak (Children's Privacy)

Aplikasi Poisy Kasir dirancang khusus untuk keperluan manajemen usaha dan operasional kasir bisnis. Aplikasi ini TIDAK ditujukan untuk anak-anak di bawah usia 13 tahun (atau 16 tahun di wilayah hukum tertentu). Kami tidak secara sengaja mengumpulkan atau meminta data pribadi dari anak-anak. Jika kami mengetahui bahwa kami tidak sengaja mengumpulkan data dari anak di bawah umur, kami akan segera menghapus data tersebut dari server kami.

⚖️ 9. Hak-Hak Pengguna

Sebagai pengguna, Anda memiliki hak-hak berikut terkait data Anda:

  • Hak Akses & Portabilitas: Memeriksa dan mengunduh laporan data penjualan serta katalog toko Anda kapan saja dari aplikasi.
  • Hak Perbaikan (Koreksi): Mengubah atau memperbarui informasi profil, data toko, dan katalog barang jika terdapat kekeliruan.
  • Hak Penghapusan: Meminta penghapusan akun dan data pribadi sebagaimana dijelaskan pada Seksi 7.
